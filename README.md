# 소개 및 조작법

## 소개
- 3인칭 TPS 기반의 로봇 액션 플랫포머입니다.  
- Stage1/Stage2 중 하나를 선택해 플레이하며,
- GoalZone에 도달하면 **MISSION CLEAR**, 
- HP가 0이 되거나 DeadZone에 떨어지면 **GAME OVER**입니다.  
- 일시정지(ESC/P)와 결과 화면(재시작/타이틀 복귀)까지 한 판 루프가 완성되어 있습니다.

## 조작법
조작법(키 안내)
- **이동**: WASD  
- **시점(카메라)**: 마우스 이동  
- **점프**: Space  
- **대시**: Left Shift  
- **조준 모드 토글**: Tab  
- **사격(단발)**: 마우스 좌클릭(LMB)  
- **연사(누르고 있는 동안)**: 마우스 우클릭(RMB)  
- **재장전**: R  
- **일시정지/재개**: ESC 또는 P  
  - Pause 메뉴: Resume / Controls / Restart / Return To Title

## 승/패 조건
- **MISSION CLEAR**: GoalZone 도달  
- **GAME OVER**: HP 0 또는 DeadZone 진입
